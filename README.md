# Expression AST

Olá! 👋 Este projeto foi desenvolvido durante o 4º período para a disciplina de Teoria de Linguagens e Compiladores.

Ele consiste em um analisador léxico e sintático capaz de processar expressões matemáticas simples e gerar representações visuais em forma de árvores sintáticas (utilizando a ferramenta Graphviz / dot).

## Tecnologias Utilizadas
- C
- Flex (Análise Léxica)
- Bison (Análise Sintática)
